# 💌 寄语：写在前面的话

> 这一部分是往届学长学姐的“肺腑之言”，如果你正感到迷茫或焦虑，希望它能给你一些力量。

如今，一纸名校学历早已不再是保你一生顺风顺水的“免死金牌”。有时候你会觉得，现行的高校体制与其说是“培养”，不如说是“筛选”。在有限的资源池里，大家似乎被迫成了养蛊场里的虫，在保研和绩点的压迫下人人自危，甚至把班级变成了一片“黑暗森林”——每个人都端着猎枪，生怕别人抢走了自己的蛋糕，一旦感受到威胁就下意识地想要防范甚至倾轧。

但作为过来人，我们真心想劝你：**请尽早放下狭隘的“优绩主义”，跳出这种零和博弈的内耗。**

别死盯着班里那几个名额卷自己人。**要去卷，就去卷外校的，去就业市场上卷工程能力，去科研里卷创新**。请记住，经历了高中严苛试炼并考入这里的你们，凭借自身的自觉性与聪明才智，**绝对足以胜任世界上任何一所顶尖名校的培养**。写下这篇指南的初衷，就是帮你们驱散前期的迷茫，让你们在毕业时能实打实地带走一身真本领，而不是在随波逐流中蹉跎了四年，最终在专业技能和通识教育上两手空空。

在互联网信息极度平权的今天，**“自学能力”才是你最强大的武器**。任何一门让你痛不欲生的课程，都能在 B站、Coursera 或全球顶尖高校的公开课里找到最优秀的讲师为你重新讲解。不要把希望全寄托在常规课堂上，要把大学当成一个巨大的资源库——学会早日自学。而这里的教授，与其说是你的“授课老师”，不如说是你迈入科研领域的**“引路人”**和项目资源的**“提供者”**。

>“如果你不知道为什么要上大学，请停下来想一想，接下来的四年你究竟想带走什么？”

这份指南只能在专业技能、选课避坑和升学就业上，给你们提供一点微笑的纪念公演。但关于人生旷野上的其余部分——如何寻找热爱、如何建立健全的人格、如何面对失败，这个我也不知道。

最后，由于我们自动化全英文班绝大多数人的最终去向是海外深造，而不是在这小小的象牙塔内。因此，强烈建议学弟学妹们**相互扶持，打通彼此间的信息差**。申请季的资料、面经、实验室的红黑榜，考试的课程资料，请毫无保留地共享。

请始终记得：一旦脱离了学校现有的评价体系，在未来的国际赛道和工业界里，坐在你身边的同学不会是你的竞争对手，而是你最宝贵的**校友人脉与合作伙伴**。

**祝你们都能找到自己的方向，前程似锦，顶峰相见！**
